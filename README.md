# 🎬 CineVerse AI - Movie Recommendation System

## 🚀 Project Overview

CineVerse AI is a full-stack movie recommendation system that suggests movies based on user input using Machine Learning and NLP techniques.

---

## ✨ Features

* 🎯 Smart Movie Recommendation (TF-IDF + Cosine Similarity)
* 😊 Sentiment Analysis on movie reviews
* 📊 EDA Dashboard with charts
* 🔮 Movie Success Prediction (ML Model)
* 🎬 Interactive Home Page with movie posters
* 🔗 Auto recommendation on poster click

---

## 🛠️ Technologies Used

* Frontend: HTML, CSS, JavaScript
* Backend: Python, Flask
* ML Libraries: Scikit-learn, Pandas, NumPy
* APIs: OMDB API

---

## 🧠 Machine Learning Techniques

* TF-IDF Vectorization
* Cosine Similarity
* Decision Tree Classifier

---

## 📂 Project Structure

* index.html (Home Page)
* recommend.html (Recommendation)
* sentiment.html (Sentiment Analysis)
* eda.html (Charts & Analysis)
* predict.html (Prediction)
* api.py (Backend)
* recommendation.py (Recommendation Logic)
* sentiment.py (Sentiment Model)
* eda.py (Data Analysis)

---

## ▶️ How to Run

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run backend:
   python api.py
4. Open index.html in browser
